# Competitive Programming Code Library

This repository contains necessary codes and algorithms which are required more often in my competitive programming (CP) journey.

## 📌 Contents

Currently, the library includes the following utilities and algorithms:

* **`maximum_subarray_sum.cpp`**: Finds the contiguous subarray within a one-dimensional array of numbers that has the largest sum. Implements the prefix sum approach to solve it efficiently in $O(N)$ time.
* **`bitwise_operation.cpp`**: A collection of common bitwise operations and their implementations, including checking the $k$-th bit, setting/unsetting bits, toggling bits, counting set bits, and checking if a number is a power of 2.
* **`Binary_to_Decimal.cpp`**: Converts a given binary number to its decimal equivalent.
* **`Decimal_to_Binary.cpp`**: Converts a given decimal number to its binary representation.
* **`Power_of_two.cpp`**: Contains a standard method to check whether a given integer is a power of 2.

## 🚀 Usage

You can compile and run any of the C++ files using a standard C++ compiler like `g++`.

To compile a file, run the following command in your terminal:

```bash
g++ filename.cpp -o filename
```

Then, run the executable:

```bash
# On Windows
filename.exe

# On Linux/macOS
./filename
```

For instance, to run the bitwise operations code on Windows:

```bash
g++ bitwise_operation.cpp -o bitwise_operation
bitwise_operation.exe
```
