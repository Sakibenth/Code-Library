# Competitive Programming Code Library

This repository contains necessary codes and algorithms which are required more often in my competitive programming (CP) journey.

## 🚀 Usage

You can compile and run any of the C++ files using a standard C++ compiler like `g++`.

To compile a file, run the following command in your terminal:

```bash
g++ filename.cpp -o filename
```

Then, run the executable:

```bash
# On Windows
filename.exe

# On Linux/macOS
./filename
```

For instance, to run the bitwise operations code on Windows:

```bash
g++ bitwise_operation.cpp -o bitwise_operation
bitwise_operation.exe
```
